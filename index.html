Generowanie kluczy

  1. Wybieramy losowo dwie liczby pierwsze p i q
  2. n = p * q
  3. Ø = (p - 1)(q - 1) // tocjent
  4. e gdzie NWD(e, Ø (n)) oraz e < n
  5. d * e % Ø (n)) == 1, d < Ø (n)

Generowanie kluczy

  1. p = 11 q = 17
  2. n = 11 * 17 = 187
  3. Ø = (11 - 1)(17 - 1) = 160
  4. e = 11, dla NWD(11, Ø (187) == 1
  5. 131 * 11 % 160 = 1
